## git status
**git status** - эта команда выводит состояние изменений: не отслеживаются, изменены или подготовлены. Также позволяет узнать, какие изменения необходимо зарегистрировать в Git (при необходимости — отменить).
<!--синтаксис-->
_Синтаксис:_
```
git status
```
